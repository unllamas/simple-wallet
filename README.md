## üöÄ Getting Started

##### 1. Clone proyect:

```bash
git clone ...
cd ...
yarn
```

##### 2. Create `.env.local` file:

```bash
INFURA_TOKEN_API = 'API KEY from Infura'
CRYPTO_SECRET_KEY = 'KEY random for encrypt/decrypt'
```

##### 3. Run the proyect:

```bash
yarn run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
